# Aplicação para agendamento e reserva de salas e veiculos - Suprimentos

VueJs 3 + Vuetify 3

## Subir aplicação
 - Execute 'npm install' para instalar as dependencias do projeto
 - Execute 'npm run dev' para subir o projeto na porta :8080
